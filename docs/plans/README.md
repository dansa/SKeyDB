# Active Plans

This folder is for active implementation plans only.

- New work starts here.
- Shipped or abandoned plans move to `../archive/plans/`.
- Notes and status docs belong in `../notes/`, not here.
- Start from `../templates/plan-template.md`.

## Required maintenance while active

- Keep `Status` accurate: `Draft`, `In progress`, `Blocked`, or `Done`.
- Update `Last updated` whenever a meaningful work session changes the plan state.
- Keep `Progress Snapshot` current instead of leaving stale checklists or old blockers behind.
- If a plan becomes historical rather than active, move it to `../archive/plans/`.
